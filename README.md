# AI Powered Interior Design Website

An AI-powered full-stack web application that helps users redesign rooms using Artificial Intelligence, 3D visualization, and smart product recommendations.

---

## 🚀 Features
- AI-based room analysis
- AI-generated interior inspirations
- 3D furniture visualization using Three.js
- Product recommendations based on room size
- Pricing calculator
- Appointment booking system
- User authentication and dashboard

---

## 🛠️ Tech Stack

| Layer | Technologies |
|---|---|
| Frontend | React.js, CSS, Three.js |
| Backend | Flask (Python) |
| Database | SQLite, JSON |
| AI/ML | LLaVA (Ollama), Pollinations AI |
| Cloud Storage | Cloudinary |
| APIs | Unsplash API, RazorPay API |
